# Fondations/Couleurs de Statut

## Neutre

Les couleurs neutres sont utilisées pour des statuts informatifs ou neutres.

<div style={{ display: 'flex', flexWrap: 'wrap', gap: '20px', marginBottom: '40px' }}>
  {Object.entries(tokens.alerts.status_colors_token).map(([key, value]) => {
    if (key.startsWith('status-neutral')) {
      return (
        <div
          key={key}
          style={{
            backgroundColor: value,
            width: '100px',
            height: '100px',
            display: 'flex',
            alignItems: 'center',
            justifyContent: 'center',
            color: '#fff',
            borderRadius: '4px',
            fontSize: '14px',
            textAlign: 'center',
            fontWeight: 'bold',
          }}
          onClick={() => navigator.clipboard.writeText(value).then(() => alert(`Copied: ${value}`))}
        >
          <p>{key}</p>
        </div>
      )
    }
    return null
  })}
</div>

# Fondations/Typographie

## Poids des Polices

Les différents poids de polices disponibles dans notre système de design.

<div
  style={{
    display: 'flex',
    flexDirection: 'column',
    alignItems: 'center',
    justifyContent: 'center',
    padding: '2rem',
    fontFamily: 'mont',
  }}
>
  <h3 style={{ marginBottom: '3rem' }}>Poids des Polices</h3>
  <table style={{ width: '100%', maxWidth: '800px', borderCollapse: 'collapse', textAlign: 'left', margin: '0 auto' }}>
    <thead>
      <tr>
        <th>Token</th>
        <th>Valeur</th>
        <th>Exemple</th>
      </tr>
    </thead>
    <tbody>
      {Object.entries(tokens.typography.global).map(([key, value]) => {
        if (key.includes('font-weight')) {
          return (
            <tr key={key}>
              <td>{key}</td>
              <td>{value}</td>
              <td style={{ fontWeight: value, padding: '0.5rem' }}>Texte d'exemple</td>
            </tr>
          )
        }
        return null
      })}
    </tbody>
  </table>
</div>

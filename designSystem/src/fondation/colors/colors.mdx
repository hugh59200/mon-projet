# Fondations/Couleurs

## Primaire

Les couleurs primaires sont utilisées pour les éléments interactifs et les call-to-action.

<div style={{ display: 'flex', flexWrap: 'wrap', gap: '20px', marginBottom: '40px' }}>
  {Object.entries(tokens.colors.colors_token).map(([key, value]) => {
    if (key.startsWith('primary')) {
      return (
        <div
          key={key}
          style={{
            backgroundColor: value,
            width: '100px',
            height: '100px',
            display: 'flex',
            alignItems: 'center',
            justifyContent: 'center',
            color: '#ccc',
            borderRadius: '4px',
            fontSize: '14px',
            textAlign: 'center',
            fontWeight: 'bold',
          }}
          onClick={() => navigator.clipboard.writeText(value).then(() => alert(`Copied: ${value}`))}
        >
          <p>{key}</p>
        </div>
      )
    }
    return null
  })}
</div>

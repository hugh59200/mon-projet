# Déploiement des Fonctions Supabase

## 1. S'assurer d'être connecté à ton projet Supabase

supabase login

## 2. Déployer la fonction order-status-update

supabase functions deploy order-status-update

## 3. Déployer la fonction order-confirmation

supabase functions deploy order-confirmation

## 4. Déployer toutes les fonctions

supabase functions deploy \
 order-status-update \
 order-confirmation \
 delete-account

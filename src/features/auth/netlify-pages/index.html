Politique de confidentialité - Fast Peptides

Fast Peptides respecte la vie privée des utilisateurs. Les données personnelles collectées (e-mail et informations de profil) sont utilisées uniquement pour permettre l’accès et l’authentification sur notre plateforme.

Données collectées

  • Adresse e-mail
  • Identifiants de connexion fournis par Google, GitHub ou Facebook

Utilisation des données

Vos données sont utilisées uniquement pour :

  • Créer un compte utilisateur
  • Permettre la connexion sécurisée
  • Envoyer des notifications liées à l’accès à la plateforme

Conservation

Les données sont conservées tant que votre compte est actif.

Suppression des données

Vous pouvez demander la suppression de votre compte et de vos données personnelles à tout moment : support@fastpeptides.com
